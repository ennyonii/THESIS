*Hemigrammus marginatus* Relational Dataset



This dataset was used as part of a Master's thesis in biological research, focusing on the trophic ecology of *Hemigrammus marginatus*. It was originally developed to 

The data has been restructured from its original wide-format spreadsheet into a normalized relational database to facilitate advanced ecological analysis, such as dietary shifts across biomes and rainfall gradients.



The database is composed of five interconnected tables, allowing for efficient querying of environmental, specimen-level, and dietary data.



1\. sites.csv

Contains the geographic and ecological context for sampling.



2\. sampling.csv

Contains information on the sampling events.



3\. fish info.csv

Contains data on every individual fish collected.



4\. diet info.csv

Contains all identified food items (prey items).



5\. relationship.csv

Contains non-zero dietary entry to match individual fish
